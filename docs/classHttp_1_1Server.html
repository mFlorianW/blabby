Detailed Description

The Http::Server can receive HTTP requests from network devices.

Note
The Http::Server should only be used in LAN and known environments.

void Http::Server::addRoute ( QByteArray const &  route,
RouteCallback const &  routeCallback 
)
noexcept

Adds a route the server. The the passed Http::ServerRequest::RouteCallback is called when a request with route is received.

Parameters
routeThe route that shall be registered.
routeCallbackThe call that shall be called when a Request for that route is received. A callback should return the true when the Http::ServerRequest is completly handled, returning false from this function will forward the Http::ServerRequest to the Http::ServerRequest::handleRequest function.
Note
Calling the route function multiple time with the same route will overwrite previous added route.
