◆ initialize()

void Multimedia::Renderer::initialize ( )
noexcept

Disabled copy and move semantic Initializes the Mulitmedia::Renderer. During the initialization the fetches all informations like, supported protocols for working properly. Without calling the initialization the Multimedia::Renderer does nothingl. The signal Mulitmedia::Renderer::initializationFinished is emitted on success. The signal Mulitmedia::Renderer::initializationFailed is emitted with a the error Information.

◆ playback()

void Multimedia::Renderer::playback ( Item const &  item)
noexcept

Starts the playback of the passed Multimedia::Item. First checks that both Multimedia::Renderer and Multimedia::Item are compatible Afterwards it calls the setAvTransportUri on the Renderer. The signal Multimedia::Renderer::playbackStarted is emitted on success. The signal Multimedia::Renderer::playbackFailed with the error information is emitted on failure.
