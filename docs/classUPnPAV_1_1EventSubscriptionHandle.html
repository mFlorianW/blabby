Detailed Description

A UPnPAV::EventSubscriptionHandle manages an event subscription between an UPnPAV device and the control point. The UPnPAV::EventSubscriptionHandle notifies about the event subscription, incoming events and event timouts. Every concret UPnPAV::EventBackend must implement it's own UPnPAV::EventSubscriptionHandle because the UPnPAV::EventBackend and UPnPAV::EventSubscriptionHandle are very tidly coupled. The implementations in the UPnPAV::EventSubscriptionHandle are useable in every other concrete UPnPAV::EventSubscriptionHandle
