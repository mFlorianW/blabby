Detailed Description

Base class for every media source. Each MediaSource shall have a name, optional icon. A MediaSource is a navigatable source of media items. Concret subclasses must implement the navigateTo(QString) function. Subclasses that are not navigatable, all playable items are on the root layer. Those subclasses don't need to overwrite the default implemenation.

◆ navigateTo()

void Multimedia::Source::navigateTo ( QString const &  path)
virtualnoexcept

The source shall navigate to the given path after succesful navigation the mediaItems() must be updated. If it fails to navigate to the given path nothing shall happen. This navigateTo(QString) must only be implemented when the source navigatable. The default implementation does nothing. The source shall emit the navigationFinished(QString) singal when the navigation is succesful finished.

Parameters
pathThe target path to navigate to.
