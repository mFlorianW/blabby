Detailed Description

An Item is an element of a Multimedia::MediaSource. Basically an Item can have two roles. The first role is the container role and the seconde role is a concrete media item. The container role is used e.g. folder albums etc. Create the Multimedia::Item instance with Multimedia::ItemBuilder.

◆ supportedTypes()

QStringList const & Multimedia::Item::supportedTypes ( ) const
noexcept

Gives the supported types of the Item. The each type consists of 4 parts separated by a ":". The parts have the following meaning: <protocol>:<network>:<contentFormat>:<additionalInfo> Format is defined in the UPnPAV ConnectionManager specification. For furhter information read the specification. The supported types are used to check if a Multimedia::Renderer is able to play a Multimedia::Item.
