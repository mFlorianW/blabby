Starts the unsubscribe process for this subscription. This function should never be called directly because the function is called by central UPnPAV::EventBackend cache, when a subscription is not longer needed. The signal UPnPAV::EventSubscriptionHandle::unsubscribed is emitted when unsubscribed.
