Detailed Description

The UPnPAV::EventBackend is the interface for a UPnPAV::MediaDevice based classes for receiving and subscribing UPnPAV events. The UPnPAV::EventBackend only handles the communication parts of the UPnPAV event mechanisim these are things like sending the subscription requests, receiving subscribtion status and to provide the callback for the UPnPAV devices to send events to a control point. Every property change and managing the life of the subscription is done via the EventSubscriptionHandle.
