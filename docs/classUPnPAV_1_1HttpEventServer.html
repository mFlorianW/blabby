Detailed Description

The HttpEventServer is the communication endpoint for UPnPAV devices which send events for subscribed events. There is only one instance in per process and all the UPnPAV::HttpEventBackends are using the singleton instance. It's possible to configure the @HttpEventServer IP address and port. The configuration is done with environment variables: "BLABBY_EVENT_SERVER_IP" and "BLABBY_EVENT_SERVER_PORT"
